## 🚀 Featured Projects

### 🛍️ [E-Commerce SPA](https://github.com/MrBlueSparks/COMP3512-assign2)

A fully client-side single-page application for browsing and purchasing products — no frameworks, just vanilla JavaScript.

- Filter & sort products by gender, category, colour, and size
- Persistent shopping cart via the `localStorage` API
- Dynamic shipping calculator and full checkout flow
- Styled with **Tailwind CSS** and deployed via **GitHub Pages**

**Tech:** Vanilla JavaScript · Tailwind CSS · localStorage API · HTML/CSS

### 📈 [Stock Portfolio Tracker](https://github.com/MrBlueSparks/COMP3512-assign1)

A server-side web app for viewing and managing customer stock portfolios backed by a relational database.

- Browse companies and their historical price data
- View per-customer portfolio summaries with live valuations
- RESTful JSON API endpoints for companies, portfolio, and price history
- Built with **PHP + PDO** talking to a **MySQL** database

**Tech:** PHP · MySQL · PDO · REST API · HTML/CSS
